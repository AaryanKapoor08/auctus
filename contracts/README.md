# Shared Types

These TypeScript files define the shared shapes used across the app:

- `role.ts` - account roles and default routes
- `profile.ts` - base profile and role-specific profile shapes
- `session.ts` - server and client session shapes
- `funding.ts` - funding rows, summaries, preferences, and query signatures
- `route-policy.ts` - route access policy types

The `@contracts/*` alias points here from `tsconfig.json`.
